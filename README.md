# Listener Perceptions of Accented Synthetic Speech

**Analyzing the Impact of L1**

Published at TABU DAG 2025

## Abstract

This study investigates how listeners' first language (L1) affects perceptions of pleasantness and trustworthiness in accented synthetic English speech.
